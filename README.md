# HR Attrition Analysis Dashboard (Tableau)

## Project Overview

This project features an **interactive HR Attrition Analysis Dashboard built using Tableau** to analyze employee attrition patterns and identify key factors influencing workforce turnover. The dashboard converts raw HR data into actionable insights by examining attrition across **departments, job roles, demographics, job satisfaction, salary levels, and tenure**, enabling data-driven HR decisions.

## Live Dashboard

**Tableau Public:**
(https://public.tableau.com/app/profile/g.suhas.kartik/viz/HRanalytics_17638835533920/AttritionDashboard)

## Key Insights

- Overall employee attrition rate
- Department-wise and job role-wise attrition trends
- Age and gender-based attrition analysis
- Impact of salary, job satisfaction, and tenure on attrition
- Identification of high-risk attrition segments

## Tools Used

- Tableau
- Excel / CSV Dataset
- Data Cleaning & Exploratory Data Analysis (EDA)

## Business Use Case

Helps HR and leadership teams monitor attrition trends, identify high-turnover roles, understand key drivers of attrition, and support data-driven workforce planning and retention strategies.
